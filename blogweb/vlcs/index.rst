Velodyne Code Sprint
--------------------

Velodyne is sponsoring work on a Plug-n-Play interface for the Velodyne HDL
series to make these sensors much easier to use and the high-density point
clouds easily accessible by developers.

.. image:: images/pcl-vlcs.png
   :align: center
   :width: 600

For a complete list of all the present and past PCL code sprints please visit http://www.pointclouds.org/blog.
