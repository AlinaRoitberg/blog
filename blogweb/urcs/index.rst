Urban Robotics Code Sprint
--------------------------

Urban Robotics is sponsoring the integration and extension of their highly
scalable, spatially searchable and colorized 3D octree-based point cloud format
in PCL, in addition to the development of out-of-core technologies based on
octree techniques for large scale point cloud datasets.

.. image:: images/pcl-urcs.png
   :align: center
   :width: 600

For a complete list of all the present and past PCL code sprints please visit http://www.pointclouds.org/blog.
