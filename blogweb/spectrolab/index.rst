Spectrolab Code Sprint
----------------------

Spectrolab, a Boeing company, is sponsoring work on the development of 3D
Viewer Software in PCL, as well as an advanced sensor grabber for Spectrolab's
SpectroScan3D LIDAR imager.

.. image:: images/spectrolab.jpg
   :align: center

For a complete list of all the present and past PCL code sprints please visit http://www.pointclouds.org/blog.
